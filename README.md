* PreAssessmentTest – React Native App
    * A simple pre-assessment app built with React Native using NativeWind (Tailwind CSS for RN), SVG support, and React Navigation.
    * This app includes user login, signup, authentication context, and protected navigation.


* Features
  * Splash Screen with automatic login detection
  * Login & Signup with form validation
  * Auth Context with persistent storage via AsyncStorage
  * Protected Routes using React Navigation
  * Tailwind CSS (NativeWind) support
  * SVG + PNG asset handling
  * Reusable components (Text, Input, Button)

* Project Structure

├── global.css
├── package.json
├── /src
│   ├── /components
│   ├── /context
│   ├── /screen

1. Clone the Repository
git clone https://github.com/Eleyeen/Pre-assessment.git
cd PreAssessmentTest

2. Install Dependencies
   npm install
   # or
   yarn install

3. Run Metro
   npm start
   # or
   npx react-native start
   
4. Run on Android
   npx react-native run-android

5. Run on iOS (Mac only)
   npx react-native run-ios

* Tech Stack
  * React Native 0.76.7
  * React 18.3.1
  * React Navigation
  * AsyncStorage
  * Tailwind CSS (NativeWind)
  * react-native-svg & svg-transformer
  * TypeScript Ready
